# New Agent App

This application demonstrates connecting to Azure OpenAI / Microsoft AI Foundry using C#, .NET, and the Microsoft Agent Framework.

## Requirements
- .NET 8.0
- Azure OpenAI Endpoint (from Microsoft AI Foundry)
- Azure OpenAI API Key

## How to Run
```bash
export AZURE_OPENAI_ENDPOINT="https://your-resource-name.openai.azure.com"
export AZURE_OPENAI_API_KEY="your-api-key"

cd NewAgentApp
dotnet run
```
